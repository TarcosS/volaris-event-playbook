# Tool Guides

Step-by-step guides for every tool coaches will use and teach during the event. Each guide is copy-paste ready and written for live event use.

| Guide | Covers |
|-------|--------|
| [Copilot Chat](copilot-chat-guide.md) | IDE Chat, @workspace, slash commands, audit & documentation prompts |
| [Copilot Instructions](copilot-instructions-guide.md) | `.github/copilot-instructions.md`, prompt files, custom instructions |
| [Azure](azure-guide.md) | Portal, `az` CLI, login, RBAC, resource inspection |
| [CLI Tools](cli-tools-guide.md) | `gh`, `az`, `git`, `npm`/`dotnet`/`python` CLIs |
| [MCP](mcp-guide.md) | Model Context Protocol servers, VS Code config, audit usage |
| [Skills](skills-guide.md) | Extensions, `.agents/skills/`, custom chat participants |
| [SDK](sdk-guide.md) | Azure SDK auth, error handling, retry patterns for auditing |

## When to Use

- **Day 1 setup** — Azure Guide, CLI Tools Guide
- **Day 2 documentation** — Copilot Chat, Copilot Instructions
- **Day 3 auditing** — MCP, Skills, SDK, Copilot Chat
- **Day 4 triage** — Copilot Chat, CLI Tools (gh issues)
- **Troubleshooting** — All guides as needed
