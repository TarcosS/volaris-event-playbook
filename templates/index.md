# Templates

Copy-paste templates for all event outputs. Each template is a standalone Markdown file — copy the fields, fill in the blanks.

| Template | Used In |
|----------|---------|
| [Audit Finding](audit-finding-template.md) | Code & Infrastructure Audit session |
| [GitHub Issue](github-issue-template.md) | Ticket Triage session |
| [Session Output](session-output-template.md) | End of every session |
| [Participant Diagnosis Sheet](participant-diagnosis-sheet.md) | Day 1 diagnosis |
| [Risk Register](risk-register-template.md) | Advanced path audit output |

## Usage

1. Open the relevant template
2. Copy the Markdown content
3. Paste into your document, issue, or notes
4. Fill in each field with evidence-based content
5. Review before submitting

> **Tip:** All templates are also collected in [Output Templates](../docs/11-output-templates.md) with descriptions and usage guidance.
